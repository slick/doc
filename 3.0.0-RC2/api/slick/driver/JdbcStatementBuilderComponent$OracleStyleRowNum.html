OracleStyleRowNum

trait OracleStyleRowNum extends JdbcDriver.QueryBuilder

QueryBuilder mix-in for Oracle-style ROWNUM (applied before ORDER BY and GROUP BY) instead of the standard SQL ROWNUMBER().
