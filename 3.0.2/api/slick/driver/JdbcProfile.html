trait JdbcProfile extends SqlProfile with JdbcActionComponent with JdbcInvokerComponent with JdbcInsertInvokerComponent with JdbcExecutorComponent with JdbcTypesComponent with JdbcModelComponent

A profile for accessing SQL databases via JDBC. All drivers for JDBC-based databases implement this profile.
