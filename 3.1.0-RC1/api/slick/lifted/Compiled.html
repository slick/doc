Compiled

Related Docs: object Compiled | package lifted

sealed trait Compiled[T] extends AnyRef

A possibly parameterized query that will be cached for repeated efficient execution without having to recompile it every time. The compiled state is computed on demand the first time a Cached value is executed. It is always tied to a specific driver.

Cached forms a limited monad which ensures that it can only contain values that are Compilable.
